# homebrew-luff

Homebrew tap for [luff](https://github.com/serhiitroinin/luff) CLI tools.

## Install

```bash
brew install serhiitroinin/luff/todo
```

Or tap first:

```bash
brew tap serhiitroinin/luff
brew install todo
```

## Available tools

| Formula | Description |
|---------|-------------|
| `todo`  | Task management CLI (Todoist) |
